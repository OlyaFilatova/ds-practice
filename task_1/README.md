# Task 1

Explores following classification models with use of a publicly available MNIST dataset:

1) Random Forest;

2) Feed-Forward Neural Network;

3) Convolutional Neural Network;

## To run project

Install Python 3.

Open task_1 directory in the terminal.

Run following commands:

`python3 -m venv venv`

`source venv/bin/activate`

`pip install -r requirements.txt`

`python3 main.py`
